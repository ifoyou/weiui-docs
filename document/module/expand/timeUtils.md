# timeUtils

> 需要加载的模块

```js
const weiui = weex.requireModule('weiui');
```

## 获取当前毫秒时间戳
```js
/**
* @参数一      固定值
* 
* @返回 Number
 */
let variable = weiui.timeUtils('getNowMills')
```

## 获取当前时间字符串
```js
/**
* @参数一      固定值
* 
* @返回 String
 */
let variable = weiui.timeUtils('getNowString')
```

## 获取当前 Date
```js
/**
* @参数一      固定值
* 
* @返回 String
 */
let variable = weiui.timeUtils('getNowDate')
```
