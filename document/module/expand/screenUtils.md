# screenUtils

> 需要加载的模块

```js
const weiui = weex.requireModule('weiui');
```

## 获取屏幕的宽度（单位：屏幕像素）
```js
/**
* @参数一      固定值
* 
* @返回 Number
 */
let variable = weiui.screenUtils('getScreenWidth')
```

## 获取屏幕的高度（单位：屏幕像素）
```js
/**
* @参数一      固定值
* 
* @返回 Number
 */

```
