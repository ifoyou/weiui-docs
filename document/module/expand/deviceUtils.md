# deviceUtils

> 需要加载的模块

```js
const weiui = weex.requireModule('weiui');
```

## 判断设备是否 rooted
```js
/**
* @参数一      固定值
* 
* @返回 true|false
 */
let variable = weiui.deviceUtils('isDeviceRooted')
```

## 获取设备系统版本号
```js
/**
* @参数一      固定值
* 
* @返回 Number
 */
let variable = weiui.deviceUtils('getSDKVersionCode')
```

## 获取设备系统版本名称
```js
/**
* @参数一      固定值
* 
* @返回 String
 */
let variable = weiui.deviceUtils('getSDKVersionName')
```

## 获取设备 AndroidID
```js
/**
* @参数一      固定值

* @返回 String
```

